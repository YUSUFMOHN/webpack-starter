# Webpack - Simple Example Project
Simple example project for Webpack. Adds the [Babel loader](https://github.com/babel/babel-loader) and two Webpack plugins:
1. [HTML Webpack Plugin](https://github.com/jantimon/html-webpack-plugin)
2. [Preload Webpack Plugin](https://github.com/GoogleChrome/preload-webpack-plugin)

The bundled files that will be served to the browser can be found in the `dist` directory.
